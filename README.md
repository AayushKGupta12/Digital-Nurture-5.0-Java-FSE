## 📌 About This Repository

Welcome to my **Cognizant Digital Nurture 5.0 Java FSE Repository**.

This repository documents my complete learning journey throughout the **Cognizant Digital Nurture 5.0 (GenC Next)** training program. It is designed to help trainees, students, and developers understand Java Full Stack concepts through practical hands-on exercises.

### It includes:

- ✅ Weekly Hands-on Exercises
- ✅ Java Fundamentals
- ✅ Object-Oriented Programming (OOP)
- ✅ Collections Framework
- ✅ Exception Handling
- ✅ Maven Projects
- ✅ Spring Core
- ✅ Spring Boot
- ✅ Spring REST APIs
- ✅ Spring Data JPA
- ✅ Hibernate ORM
- ✅ Microservices
- ✅ Angular
- ✅ Git & GitHub
- ✅ SQL
- ✅ JUnit & Mockito
- ✅ Design Patterns
